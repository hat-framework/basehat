Tipos Reconhecidos no array dos models:

  • type

    tipo do banco de dados: int, float, varchar ...

  • size

    tamanho máximo da string na aplicação

  • notnull

    define se deve ou não ser preenchido

  • especial

    tipo especial atribuido: monetary, intSlider, etc

  • fkey

    nome do model do qual é carregada a chave estrangeira
